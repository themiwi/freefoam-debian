[[chap_postProcessing]]
== Post-processing
(((post-processing)))

This chapter describes options for post-processing with {project}. {project} is
supplied with a post-processing utility `para` that uses 'ParaView', an open
source visualisation application described in <<sec_paraFoam>>.

Other methods of post-processing using third party products are offered,
including 'EnSight', and the post-processing supplied with 'Fluent'.
